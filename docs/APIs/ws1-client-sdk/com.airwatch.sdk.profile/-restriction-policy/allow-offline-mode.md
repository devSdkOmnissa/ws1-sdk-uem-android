# allowOfflineMode

```kotlin

```

<div class="api-sig-types" markdown="span">[Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin-stdlib/kotlin/-boolean/index.html)</div>

Determine if the application can be used when access to network is not available.

## Return

true if application can be used when device is offline. false if application cannot be used when device is offline.
