# preventCopyAndCutActions

```kotlin
open fun preventCopyAndCutActions(): Boolean
```

<div class="api-sig-types" markdown="span">[Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin-stdlib/kotlin/-boolean/index.html)</div>

Determine if the application is allowed to use clipboard copy and cut actions.

## Return

true if use of clipboard is prohibited for the application. false if the application is allowed to use clipboard actions.
