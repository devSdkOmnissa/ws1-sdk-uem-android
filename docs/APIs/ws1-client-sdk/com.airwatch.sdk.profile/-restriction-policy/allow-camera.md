# allowCamera

```kotlin
open fun allowCamera(): Boolean
```

<div class="api-sig-types" markdown="span">[Boolean](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin-stdlib/kotlin/-boolean/index.html)</div>

Retrieve application restriction information for use of the camera. If the device restriction prohibits the use of the camera, it takes precedence over the application setting.

## Return

use of the camera is prohibited. false if use of camera is allowed.
