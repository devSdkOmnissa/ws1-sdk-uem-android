# getEulaContent

```kotlin
open fun getEulaContent(): String
```

<div class="api-sig-types" markdown="span">[String](https://developer.android.com/reference/kotlin/java/lang/String.html)</div>

Provides content text for End User License Agreement.

## Return

Non-null description to be displayed as part of End User License Agreement. This field will have empty string if there is no EULA configured or there is error receiving it.
