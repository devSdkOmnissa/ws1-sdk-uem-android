# getApplicationProfile

```kotlin
open fun getApplicationProfile(): ApplicationProfile
```

This method is used to get the Application Profile as defined at the WS1 UEM Console. 

Access Level = AW_MANAGED (for details on access permission please check [SDKManager](index.md)) 

## Return

ApplicationProfile object on success. null in case of failure.
